Про мене

Мене звати Вадим, являюсь Front End розробником. Надаю послуги з професійної розробки веб-проектів, високоякісної адаптації веб-ресурсів під будь-які розміри екранів, консультації стосовно дизайну і реалізації проекту, чіткої розробки згідно технічного завдання.

До основного стеку технологій, якими користуюсь при розробці можна віднести: HTML5, CSS3 (SCSS, SASS, FlexBox, Grid, Responsive), JS (ES5, ES6+), React (Hooks, React-router-dom), Next.js, React Native, React Navigation, Expo, Redux, TypeScript. Базові знання Bootstrap, Materialize. Досвід використання Firebase (авторизація, використання баз даних, деплой). Активне використання Gulp, Webpack
